## Full Description

```
Zendesk File Renamer automatically prefixes downloaded files with the ticket number, helping support engineers organize logs, attachments, and artifacts without manual renaming.

BEFORE: debug.log, attachment.pdf, screenshot.png
AFTER: ZD-123456_debug.log, ZD-123456_attachment.pdf, ZD-123456_screenshot.png

HOW IT WORKS
• Navigate to any Zendesk ticket
• Download attachments as usual
• Files are automatically renamed with the ticket number

FEATURES
• Automatic renaming - no extra clicks required
• Multiple format options - choose ZD-{ticket}_, [{ticket}], or {ticket}-
• Works with multiple tabs - each download gets the correct ticket number
• Toggle on/off - easily disable when not needed
• Zero configuration - works out of the box

PRIVACY
• No data collection
• No external servers
• All processing happens locally in your browser
• Open source: https://github.com/bealmot/zendesk-file-renamer

PERFECT FOR
• Support engineers managing multiple tickets
• Teams that need to organize customer logs
• Anyone tired of manually renaming Zendesk downloads

Save time. Stay organized. Download and forget about renaming.
```
